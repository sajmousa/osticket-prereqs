<h1>osTicket - Prerequisites and Installation</h1>
This tutorial outlines the prerequisites and installation of the open-source help desk ticketing system osTicket.<br />


<h2>Video Demonstration</h2>

- ### [YouTube: How To Install osTicket with Prerequisites](https://www.youtube.com)

<h2>Environments and Technologies Used</h2>

- Microsoft Azure (Virtual Machines/Compute)
- Remote Desktop
- Internet Information Services (IIS)

<h2>Operating Systems Used </h2>

- Windows 10</b> (21H2)

<h2>List of Prerequisites</h2>

- Create or have an active Azure subscription
- Create resource Group which will include a virtual network and subnet.
- Create Virtual Machine
- Install the osTicket requirements
- Install osTicket

<h2>Installation Steps</h2>

<p>
<img src="https://i.imgur.com/gja4Jjq.png" height="80%" width="80%" alt="Disk Sanitization Steps"/>
<img src="https://i.imgur.com/2cot5v9.png" height="80%" width="80%" alt="Disk Sanitization Steps"/>

<p>
To get started with installing osTicket please be sure to have an active Azure subscription. The first step is to create a resource group, click on resource group on the home page, and then it will route you into the resource group folder. Click on create and you will now be able to create a new resource group folder.
</p>
<br />

<p>
<img src="https://i.imgur.com/5mEliPn.png" height="80%" width="80%" alt="Disk Sanitization Steps"/>
</p>
<p>
To create the resource group folder, you will need to select your subscription (Please note that in order to create this resource group an ACTIVE AZURE SUBSCRITION IN NEEDED), enter a resource group name, and select the region you are in. Once all 3 steps have been completed select review and create.
</p>
<br />

To see what has been created click on the resource group folder and it will show an overview of the folder that was created.
